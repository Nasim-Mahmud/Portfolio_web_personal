Ambitious Academic Projects

    Course: CSE115 - Programming language I

    Summer 2017

    Description: The project serves as an example of the need for secure end-to-end communication. This program aims to transmit important or sensitive national security messages between authorized personnel. One must be eligible to use this program as an admin or a user. Three-step verification is used in the design of this program to guard against unauthorized access. The administrator and user can access this program by entering their username and password. After some verification, they can deliver messages to one another. The program will securely store every message and send it to its intended recipient with the appropriate security.

    Tech used: C, C++

    Course: ENV203 - Introduction to Bangladesh Geography

    Spring 2019

    Description: The expansion of urban areas and colonization have significantly impacted the land area of various regions over time. Bangladesh's rural areas have undergone numerous changes comparable to this. There is considerably less water in the area, and there are fewer green spaces. We used the Google Earth engine to create a GIS environment model to highlight the problem. NASA and the U.S. Geological Survey jointly manage earth-observing satellite missions that provide the data we used in this project (USGS LANDSAT 8 Reflectance Tier 1). By gathering data over the previous 10 years, we visualized the data using a time-lapse video.

    Tech used: C, C++

    Course: CSE323

    Fall 2019

    Description: DexBot, a disaster exploring robot designed to access difficult terrain with its unique suspension system, is a remote-controlled robot that uses an integrated GPS module and a back-as-a-service (BAAS) called Firebase to detect and locate obstacles. Capstone Design Projects

    Course: CSE299 - Junior Design Project

    Spring 2020

    Description: The goal of this project is to implement a low-cost, easily-affordable, and user-friendly hand gesture recognition system to help the elderly and disabled who are incapable of to move or use voice to communicate. The system will employ a phone-based app that enables users to call for assistance while standing far away by making a hand motion. People who cannot use a cell phone in an emergency, such as those who travel at night or live in rural areas, will benefit from this.

    Tech used: Python, OpenCV, Raspberry Pi 3 B+, Pi-camera, Firebase, Android Studio

    Project link: THE LINK

    Course: CSE499 - Senior Design Project I & II

    Summer 2021 - Spring 2022

    Description: Bangladesh is a heavily populated nation that experiences natural disasters frequently. It can be challenging and dangerous for people to explore the affected areas in these circumstances. In such cases, autonomous exploring robots can be helpful because they can search the site independently for survivors. AOAVER, which stands for Autonomous Obstacle Avoiding Exploring Robot, is the name of the robot that we developed. AOAVER can navigate static obstacles and explore locations that are inaccessible to humans.

    Tech used: Python, YDLidar X4, Raspberry Pi 3 B+, Arduino UNO, Linux OS, ROS
